# 安装

## 安装问题

- 解决Mac终端的命令行前的(base)问题
设置 conda 不自动启动 base 环境就可以了: conda config --set auto_activate_base false

如果想让 conda 自动启动 base 环境: conda config --set auto_activate_base true

conda 的环境可以用 conda env list 查看
